# Arduino for Visual Studio

Этот проект предназначен для разработки программного обеспечения для микроконтроллеров AVR с использованием WinAVR в среде Visual Studio.

## Установка WinAVR

1. **Скачайте WinAVR**:
   - Перейдите по [этой ссылке](http://sourceforge.net/projects/winavr/) для загрузки последней версии WinAVR.

2. **Установите WinAVR**:
   - Запустите скачанный инсталлятор и следуйте инструкциям на экране.
   - Рекомендуется установить WinAVR в директорию без пробелов и кириллических символов, например: `C:\WinAVR`.

## Настройка пути

После установки WinAVR вам может понадобиться добавить путь к его исполняемым файлам в переменные среды Windows:

1. Откройте "Панель управления" и перейдите в "Система".
2. Нажмите на "Дополнительные параметры системы".
3. В разделе "Переменные среды" найдите переменную `Path` и выберите "Изменить".
4. Добавьте путь к папке `bin` внутри установленной директории WinAVR (например, `C:\WinAVR\bin`).

## Перемещение папки include

Если вы хотите использовать собственные заголовочные файлы или библиотеки, переместите папку `include` в удобное для вас место и обновите пути в вашем проекте, чтобы они указывали на новое местоположение.

## Сохранение файлов без UTF-8 BOM

Для корректной компиляции файлов с помощью компилятора AVR необходимо сохранять их без BOM (Byte Order Mark). Вот как это сделать:

1. Если вы используете текстовый редактор, такой как Notepad++, выполните следующие шаги:
   - Откройте файл.
   - Перейдите в меню **Кодировка**.
   - Выберите **UTF-8 без BOM**.
   - Сохраните файл.

2. Если вы используете другой редактор, убедитесь, что он поддерживает сохранение файлов без BOM.

## Заключение

Следуя этим инструкциям, вы сможете настроить среду разработки для работы с Arduino и микроконтроллерами AVR с использованием WinAVR. Если у вас возникли вопросы или проблемы, не стесняйтесь обращаться за помощью.
